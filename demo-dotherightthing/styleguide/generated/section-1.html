1.2.0 - Breadcrumbs

Description:

Navigation list.

HTML markup:

<div class="breadcrumb">
  <h2 class="hidden">You are here:</h2>
  <ul>
    <li><a href="/">Home</a></li>
    <li><a href="/section">Section</a></li>
    <li>Page</li>
  </ul>
</div>

CSS/HTML Modifiers:

Default layout

You are here:

  • Home
  • Section
  • Page
